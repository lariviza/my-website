Theme Installation

Install WordPress

Before installing Cryptozfree you have to install wordpress. You can install it yourself or use any web host that installs wordpress for you. To know more visit WordPress.org.

Download and Unzip the theme after free purchase

1. After free purchasing the theme, you will recive an email with download link. Open this link and use the Download button at the middle of the page to download your template package.

2. You will need to extract/unzip the package to start working with files. You will find cryptozfree.zip and Documentation files.

Install Cryptozfree

1. First, go to Appearance > Themes and click Add New to add Cryptozfree Theme.

2. Next, click Upload Theme to upload the theme file you have purchased.

3. Now, click Choose File and select the cryptozfree.zip file from your pc to upload it.

4. Next, click Install Now to start the installation process.

5. Finally, when the cryptozfree.zip file is uploaded and installed completely you will see “Theme installed successfully” message. But you should not activate this one. Hence, we suggest you to install and activate the cryptoz child theme we provided for your convenience.

Demo Import

Install Plugins

1. Go to Plugins, click Begin Installing Plugins.

2. Check all plugins. Then select Install in the Bulk Actions dropdown and click Apply.

3. And finally, you will see the confirmation that all plugins are installed and activated successfully. Click Return to the Dashboard link right side of the conformation message shown.

Server Settings

Most of the time demo import fails because of server settings. Please make sure your server settings have minimum these values:

  • upload_max_filesize: 256M
  • max_input_time: 600
  • memory_limit: 256M
  • max_execution_time: 600
  • post_max_size: 512M

You can contact your hosting provider to make the changes for you. Try importing the demo again when ready.

Import Demo Data

To setup Cryptozfree same as the live demo, follow the steps:

1.First, go to Appearance > Import Demo Data and click Import Demo Data.

2. Click Import Demo Data button.

Now Click On Continue & Import button to start importation of the demo website.

3. Please wait until the demo data is imported properly because it may take a while.

4. And finally when demo data import is Import Complete you will see a confirmation message.

Theme Setup

Permalinks

Go to Settings > Permalinks and select the Post Name & Hit Save.

Update Site Address (URL)

Go to Elementor Tools and choose Replace URL

For Updating Site URL, put old url (http://cryptoz.iamabdus.com/free/) and new url (Which is Your Wordpress URL). Then hit the Replace URL button.

Delete Default Post

Go to Post. Now You Can See The Default Post Name "Hello World".

Now You Can See The Unchecked Button, Make Checked by Clicking The Unchecked Button & Select Move to Trash. Now Click Apply Button.

Customization

How to Change Logo

1. Go to Appearance > Customize > Logo

2. Click Change logo, upload and select your desired logo and hit Publish.

How to Change Site Title

Now, type in your site’s title in the Site Title input. You can also type in a Tagline below to set your site’s tagline.

How to Change Site Icon (Favicon)

Now, Click Change image and upload the image you want to use as your site icon. We suggest to use an image that has a dimension of 512 by 512 pixels.

How to Change Site Color

1. Go to Appearance > Customize > Color

Now, Change The Color As You Like.

2. If You Want to See The Color Set Click The Folder.

Now You Can See 8 Color Set. Choose the Color Set That You Liked To Use That Site. And Hit Publish Button.

How to Change Preloader Image

1. Go to Appearance > Preloader Options

2. Now Click Change image and upload the image you want to use as your as preloader.

3. Now choose the Preloader image size and background colorThen hit Publish.

4. By  Defalut Preloader is Active. But if you want to deactive, Uncheck the Enable Preloader.

How to Change Header

1. Go to Appearance >  Customizer > Header Template

2. Use Header dropdown menu to select header and hit Publish. If you want to create a new header follow How to Create New Header section below

How to Change Footer

1. Go to Appearance >  Customizer > Footer Template

2. Use Footer dropdown menu to select footer and hit Publish. If you want to create a new footer follow How to Create New Footer section below
